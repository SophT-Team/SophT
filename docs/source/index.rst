Welcome to SophT's documentation!
=================================

Scalable One-stop Platform for Hydroelastic Things (SOPHT) corresponds to the
development of flow-structure interaction simulator (2D and 3D) using immersed
boundary method, while capturing Cosserat rod dynamics using
`pyelastica`_.

.. _pyelastica: https://github.com/GazzolaLab/PyElastica
